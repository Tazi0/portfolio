Hallo, ik ben Tazio de Bruin, nu 17 jaar oud en zit op het Grafisch Lyceum Utrecht in het 2e jaar in 2wdv1-a. Ik ben geboren in Utrecht en toen ik 3 was was ik verhuist naar Emmen, ik had daar altijd al een passie voor elektrische machines en computers. Toen ik op mijn 6e naar Culemborg ben verhuist ging ik regelmatig op de computer, eerst nog online spelletjes naar Steam games gegaan. Toen was ik benieuwd hoe ik een game kon maken en zag dat je daarvoor moest coderen, ik begon op mijn 13e begonnen met coderen en heb toen van kleine login pagina's naar websites en mijn ervaring verbreid. Nu kan ik HTML, CSS, JavaScript, SQL, PHP, Lua, Git, Node.JS en Python. Ik hou van nieuwe talen leren en ben altijd open voor een uitdaging.

Mijn passie om door te gaan met coderen is dat er altijd verbetering in zit, ik wil mijn websites zo makkelijk mogelijk maken voor de bezoeker. Modern & Strak is mijn doel. Het geeft mij rust als het opgeruimd is. (en ik hou van geheimen in mijn website gooien, vind jij de 2, er zit ook decrypting bij)

HTML

Ik zie HTML niet echt als een programmerings taal omdat het geen uitdaging heeft, HTML is meer een skalet waarop de persoon wordt gemaakt. Hier doe je eigenlijk vrij weinig mee alleen informatie in stoppen en hopen dat alles er netjes uit ziet.

JavaScript

Het meeste ervaring heb ik met JavaScript, ik hou van de makkelijkheid of hoe overzichtelijk het is. Je hebt velen uitbreidingen en is het een van de populairste coderings talen die er op het moment zijn.

PHP

Ookal is PHP alleen voor back-end developen heb ik hier nogsteets veel plezier mee en heb met Casey (een goede vriend van mij in de 1e), ik heb in dat jaar ook PHP geleerd. Wat me was opgevallen is hoe het erg op JavaScript lijkt.

Node.JS

Node.JS is weer een extensie op JavaScript waar je veel aan hebt, het geeft je packages waarbij je allemaal informatie kan halen door API's te gebruiken en heb je ook erg veel aan in de web development gedeelte. Ik heb deze ervaring opgedaan toen ik bezig was met een Discord bot aan het maken was genaamt Lyfe. Waarbij ik nogsteets regelmatig aan werk met plezier voor anderen.
